# Factorial

## Compile

## a.out and factorial_dats.c

    patscc factorial.dats

## Compile C code

    gcc -I${ATSINCLUDE} -I${ATSINCLUDE}/ccomp/runtime factorial_dats.c 

## factorial_dats

    myatscc factorial.dats

## Run

    ./a.out
